# Gamification Pages Redesign v2 — Design Spec

## Overview

在现有玻璃态基础上全面升级4个 gamification 页面的视觉冲击力：更大字号、更丰富布局、更强层次感。只改前端 UI，不改后端 API。

**设计方向**：方案 A+C 混合 — Hero 驱动 + 沉浸式游戏化

---

## 1. 共享设计系统

### 1.1 字号升级

| 层级 | 当前 | 新设计 |
|------|------|--------|
| 页面标题 | `text-3xl/4xl` | `text-5xl font-black`（渐变文字） |
| 副标题/描述 | `text-sm` gray-500 | `text-lg` gray-600 |
| 统计数据 | `text-2xl/3xl` | `text-5xl font-black`（渐变数字） |
| 统计标签 | `text-sm` | `text-base font-medium` |
| 卡片标题 | `font-semibold` | `text-lg font-bold` |
| 卡片正文 | `text-sm` | `text-base` |
| 辅助文字 | `text-xs` | `text-sm` |
| 按钮文字 | `text-sm` | `text-base font-semibold` |

### 1.2 间距升级

| 区域 | 当前 | 新设计 |
|------|------|--------|
| 页面内边距 | `px-6 lg:px-12 py-8` | `px-8 lg:px-16 py-10 lg:py-16` |
| 标题到内容 | `mb-8/10` | `mb-12/16` |
| 卡片内边距 | `p-5/6` | `p-6/8/10` |
| 卡片间距 | `gap-4/5` | `gap-6` |

### 1.3 色彩增强

- 背景从单调渐变升级为**三层渐变 + 装饰光斑**（绝对定位模糊圆形）
- 每个页面有不同色调的光斑装饰
- 保留现有玻璃态 token（`bg-white/70`, `backdrop-blur-xl`, `border-white/40`）
- 卡片增加 hover 微动效（`hover:-translate-y-1` + 阴影增强）

### 1.4 新增通用视觉元素

- **GradientBlob**：绝对定位的模糊渐变圆，作为背景装饰（各页面位置/颜色不同）
- **Hero Banner**：全宽玻璃大卡 `p-10`，带发光渐变边框，水平排列核心数据
- **StatPill**：统计数字组件 — 大 emoji/图标 + `text-5xl` 渐变数字 + `text-lg` 标签

## 4. 不改的部分

- 所有后端 API（路由、服务、数据库）
- TypeScript 类型定义（`types/gamification.ts`）
- React hooks（`hooks/useGamification.ts`）
- 路由配置（`routes.tsx`）
- 业务逻辑（筛选、Tab 切换、购买、装备等）
- Loading / Error / Empty 状态的处理逻辑（只改样式）

---

## 5. 测试策略

- 现有单元测试覆盖组件逻辑，视觉改动不破坏测试
- 每个页面改完后运行 `cd frontend && npm test`
- E2E 测试验证页面导航和基本交互
- 视觉验证：手动检查4个页面在 1920px / 1440px / 375px 宽度下的表现

---

## 6. 成功标准

1. 4个页面使用一致的升级版玻璃态设计
2. Hero Banner 模式统一应用于成就/每日挑战/积分中心
3. 字号从 `text-sm` 体系升级到 `text-base` 体系，标题 `text-5xl`
4. 所有现有功能不变（筛选、Tab、购买、装备、加载更多等）
5. 前端单元测试全通过
6. TypeScript 编译无错误
7. Loading / Error / Empty 状态同步升级
